## How to Add Preferences

When a user expresses a preference, add it here in the format:

```markdown
### Category
- **Preference**: What they prefer
- **Context**: When this applies
- **Learned**: Date/Session
```

### Example

```markdown
## Code Style

### Formatting
- **Preference**: Always use trailing commas in multiline arrays
- **Context**: Both iOS and Android
- **Learned**: 2024-01-15

### Naming
- **Preference**: Use descriptive variable names over short ones
- **Context**: All code
- **Learned**: 2024-01-20
```
